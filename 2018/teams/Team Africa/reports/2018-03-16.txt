Report of OpenDevs's course on 2018-03-16


But du Jour : Contribuer au projet choisi dans le cadre du reverse engineering



La première phase est d'analyser toutes les possibilités qui nous ont été données par le responsable
du projet open source, dans notre cas pour le projet PharoJS Mr Noury nous a proposé de :

- all tests are currently green, except for 3 are marked as expected failures, and hence we need to fix them.
- Since a "security update", Safari raises an error when loading JS code generated by PharoJS. So, all tests fail.
-Currently, PharoJS works for Pharo 5. We decided to skip Pharo 6. We prefer instead to port to Pharo 7. So, plenty of work to do here.
-We also need a continuous integration server



Nous avons choisi de travailler sur les tests et d'essayer de résoudre les problèmes les empêchant de
bien fonctionner.
